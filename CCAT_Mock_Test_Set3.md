**Q3.** Change to passive: "They have been repairing the road for two days."

- A. The road has been being repaired by them for two days.
- B. The road has been repaired by them for two days.
- C. The road was being repaired by them for two days.
- D. The road is being repaired by them for two days.

**Answer: A. The road has been being repaired by them for two days.**

> **Explanation:** The active sentence is in **Present Perfect Continuous** tense. The passive formula for this tense is: Object + **has/have + been + being + V3** + by + Subject. Though this form is rarely used in everyday speech, it is grammatically correct. Option B (Present Perfect Passive) loses the continuous/ongoing aspect. Options C and D shift the tense incorrectly.

**Q4.** Identify the correct passive form of: "Let him do the work."

- A. Let the work be done by him.
- B. The work should be done by him.
- C. Let the work done by him.
- D. He should let do the work.

**Answer: A. Let the work be done by him.**

> **Explanation:** For imperative sentences beginning with "**Let**," the passive structure is: **Let + object + be + past participle**. So "Let him do the work" ‚Üí "Let **the work** be **done** by him." Option C omits "be" making it grammatically incorrect. Option B changes the modal meaning entirely.

It frames data, controls access to the physical medium, and ensures reliable node-to-node delivery. The Physical Layer handles raw bit transmission; the Network Layer handles routing.

**Q97.** In Hadoop's MapReduce, which of the following describes the Shuffle and Sort phase?

- A. It distributes data blocks across DataNodes in HDFS

- C. It compresses the final output of the Reduce phase before writing to HDFS

**Answer: B.**

**Q98.** Which of the following best describes data replication in HDFS?

- A. Replication compresses data to reduce storage usage across the cluster
- B. Replication stores multiple copies (default: 3) of each data block on different DataNodes to ensure fault tolerance and data availability
- C. Replication encrypts data before distributing it across nodes for security
- D. Replication distributes computation tasks evenly across all nodes to improve speed

**Answer: B.**

> **Explanation:** **HDFS replication** stores each data block **3 times by default** (replication factor = 3) across different DataNodes. If one DataNode fails, the data remains accessible from another replica. The **NameNode** monitors block replication and triggers re-replication if a node goes down, ensuring **fault tolerance** and **high availability**.

**Q99.** In Apache Spark, what is the difference between a transformation and an action?

- A. Transformations are eager operations that immediately compute results; actions are lazy and deferred
- B. Transformations create a new RDD from an existing one and are lazily evaluated; actions trigger the actual computation and return a result to the driver
- C. Transformations write data to disk; actions process data in memory
- D. There is no functional difference; the terms are used interchangeably in Spark

**Answer: B.**

> **Explanation:** Spark uses **lazy evaluation**. **Transformations** (e.g., `map()`, `filter()`, `flatMap()`) define a computation plan on an RDD but **do not execute immediately** ‚Äî they build a DAG (Directed Acyclic Graph). **Actions** (e.g., `count()`, `collect()`, `save()`) **trigger the actual execution** of all queued transformations and return a result to the driver program.

**Q100.** Which of the following metrics is used to evaluate the performance of a classification model in Machine Learning?

- A. Mean Squared Error (MSE)
- B. R-squared (R¬≤)
- C. F1 Score
- D. Root Mean Squared Error (RMSE)

**Answer: C. F1 Score**

> **Explanation:** The **F1 Score** is the harmonic mean of **Precision** and **Recall**: F1 = 2 √ó (Precision √ó Recall) / (Precision + Recall). It is specifically used for **classification** models, especially when class distribution is imbalanced. **MSE, RMSE, and R¬≤** are **regression** metrics used to evaluate models predicting continuous numerical values.
