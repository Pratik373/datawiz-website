**Q3.** Change to passive: "They say that honesty is the best policy."

- A. Honesty is said to be the best policy.
- B. It is said that honesty is the best policy.
- C. Honesty is told to be the best policy.
- D. Both A and B are correct.

**Answer: D — Both A and B are correct**
> Reporting verb passives allow two valid structures: the *impersonal* form — "It is said that honesty is the best policy" — and the *personal passive* — "Honesty is said to be the best policy." Option C is wrong because "told" requires an indirect object and cannot be used here.

**Q4.** Change to passive: "The sudden noise made the child cry."

- A. The child was made crying by the sudden noise.
- B. The child was made to cry by the sudden noise.
- C. Crying was made by the child due to the sudden noise.
- D. The child is made to cry by the sudden noise.

**Answer: B — The child was made to cry by the sudden noise.**
> With causative verbs (make, let, hear, see) in the active voice, the bare infinitive becomes a *to-infinitive* in the passive. So "made cry" → "was made to cry." Option D uses the wrong tense (present instead of past).

**Q52.** [Conditional Statements] What is the output of the following code?
```c
int x = 0;
if (x = 5)
    printf("True");
else
    printf("False");
```
- A. False
- B. Compilation error
- C. True
- D. Nothing

**Answer: C — True**
> `x = 5` is an *assignment*, not a comparison. It assigns 5 to `x` and evaluates to 5. Since any non-zero value is treated as `true` in C, the `if` branch executes. This is a classic C pitfall — use `==` for comparison.

**Q54.** [Functions] What is the output of the following recursive function called as `fun(3)`?

**Q64.** [Linked List] To find the middle of a linked list in a single pass, the optimal approach is:

- A. Count nodes first, then traverse to n/2
- B. Use two pointers — slow (1 step) and fast (2 steps)
- C. Use a Stack to store all nodes
- D. Use Binary Search

**Answer: B — Use two pointers — slow (1 step) and fast (2 steps)**
> The Floyd's two-pointer (tortoise and hare) technique: the slow pointer moves one node at a time, the fast pointer moves two. When the fast pointer reaches the end, the slow pointer is exactly at the middle. This achieves O(n) time and O(1) space in a single pass.

**Answer: C — AVL Tree**

**Q68.** [Time Complexity] Using the Master Theorem, what is the solution to T(n) = 2T(n/2) + O(n)?
