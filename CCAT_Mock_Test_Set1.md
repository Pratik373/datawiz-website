**Q1.** He is going _____ school _____ the morning.

- A. to / in
- B. at / on
- C. to / on
- D. in / at

**Answer: A. to / in**

> **Explanation:** We use "go **to** school" (destination) and "**in** the morning" (time of day). "At" is used for specific points in time (e.g., at 9 AM), and "on" is used for days (e.g., on Monday).

**Q2.** The book is _____ the table.

- A. in
- B. on
- C. at
- D. by

**Answer: B. on**

> **Explanation:** "On" is used to indicate something resting on a surface. "In" implies being inside something, "at" refers to a general location or point, and "by" means near/beside.

**Q3.** Change to passive voice: "Ram eats mangoes."

- A. Mangoes are eaten by Ram.
- B. Mangoes were eaten by Ram.
- C. Mangoes have been eaten by Ram.
- D. Mangoes is eaten by Ram.

**Answer: A. Mangoes are eaten by Ram.**

> **Explanation:** The active sentence is in Simple Present tense. Passive voice formula for Simple Present: Object + **is/are** + Past Participle + by + Subject. "Mangoes" (plural) takes "are," and "eat" → "eaten." Option B uses past tense; C uses present perfect; D incorrectly uses "is" with a plural noun.

---

**Q4.** Identify the passive sentence:

- A. She writes a letter.
- B. A letter is written by her.
- C. She has written a letter.
- D. She wrote a letter.

**Answer: B. A letter is written by her.**

> **Explanation:** A passive sentence has the structure: Object + auxiliary verb (is/are/was/were) + past participle + by + agent. Option B follows this structure: "A letter" (object) + "is written" (auxiliary + past participle) + "by her" (agent). Options A, C, and D are all active voice sentences.

**Answer: B. saw**

> **Explanation:** "Have" is a present perfect auxiliary verb and must be followed by the **past participle**. The past participle of "see" is **"seen,"** not "saw" (which is simple past). Correct sentence: "I have **seen** this movie before."

**Q68.** Which principle does a Stack follow?

- A. FIFO
- B. LIFO
- C. FILO
- D. Both B and C

**Answer: D. Both B and C**

> **Explanation:** A **Stack** follows **LIFO (Last In, First Out)** — the last element inserted is the first to be removed. **FILO (First In, Last Out)** is simply another name for the same principle: the first element inserted is the last to be removed. Both LIFO and FILO describe the same stack behavior, so **D** is correct.

**Q69.** Which operation adds an element to the rear of a Queue?

- A. Push
- B. Pop
- C. Enqueue
- D. Dequeue

**Answer: C. Enqueue**

> **Explanation:** In a Queue (FIFO — First In, First Out):
> - **Enqueue** adds an element to the **rear** (back).
> - **Dequeue** removes an element from the **front**.
> Push/Pop are Stack operations (not Queue).

**Q70.** In a singly linked list, the last node points to:

- A. The first node
- B. Itself
- C. NULL
- D. The middle node

**Answer: C. NULL**

> **Explanation:** In a **singly linked list**, each node has a data field and a pointer to the next node. The **last node's** next pointer is set to **NULL** to indicate the end of the list. (If it pointed to the first node, it would be a circular linked list.)

**Q71.** A binary tree allows each node to have at most how many children?

- A. 1
- B. 2
- C. 3
- D. Unlimited

**Answer: B. 2**

> **Explanation:** A **Binary Tree** is defined as a tree data structure where each node has **at most 2 children** — referred to as the left child and the right child. This "binary" (two) constraint is what distinguishes it from general trees (unlimited children) or n-ary trees.

**Q72.** Which search algorithm checks every element one by one?

- A. Binary Search
- B. Jump Search
- C. Linear Search
- D. Interpolation Search

**Answer: C. Linear Search**

> **Explanation:** **Linear Search** (also called Sequential Search) checks each element one by one from the beginning until the target is found or the list ends. Its time complexity is O(n). Binary Search requires a sorted array and uses divide-and-conquer (O(log n)); Jump Search and Interpolation Search are also more efficient variants.

**Q73.** Which sorting algorithm repeatedly swaps adjacent elements if they are in the wrong order?

- A. Merge Sort
- B. Insertion Sort
- C. Quick Sort
- D. Bubble Sort

**Answer: D. Bubble Sort**

> **Explanation:** **Bubble Sort** works by repeatedly comparing and swapping **adjacent elements** if they are in the wrong order, causing larger elements to "bubble up" to the end of the array. This process repeats for each pass until the array is sorted. Its time complexity is O(n²) in the worst case.

**Q74.** Which of the following is used to represent a Graph?

- B. Adjacency Matrix and Adjacency List
- C. Arrays only

**Answer: B. Adjacency Matrix and Adjacency List**

**Q75.** What is the time complexity of accessing an element in an array by index?

### Section 8: Operating Systems

**Q84.** Which of the following is the main purpose of an Operating System?

- A. Compile programs
- B. Manage hardware and software resources
- C. Design databases
- D. Create web pages

**Answer: B. Manage hardware and software resources**

> **Explanation:** An **Operating System (OS)** acts as an intermediary between users/applications and computer hardware. Its main functions include managing CPU (process scheduling), memory, file systems, I/O devices, and providing a user interface. Compiling programs is done by a compiler; designing databases is done by DBMS; web page creation is a development task.

---

**Q85.** In the process state model, which state indicates that a process is waiting for an event to occur?

- A. Ready
- B. Running
- C. Blocked/Waiting
- D. Terminated

**Answer: C. Blocked/Waiting**

> **Explanation:** In the OS process lifecycle:
> - **New**: Process is being created.
> - **Ready**: Process is waiting to be assigned to the CPU.
> - **Running**: Process is currently being executed by the CPU.
> - **Blocked/Waiting**: Process is waiting for an external event (e.g., I/O completion, resource availability).
> - **Terminated**: Process has finished execution.

---

**Q86.** Which scheduling algorithm gives the minimum average waiting time in most cases?

- A. FCFS
- B. Round Robin
- C. Shortest Job First
- D. Priority Scheduling

**Answer: C. Shortest Job First**

> **Explanation:** **Shortest Job First (SJF)** is proven to give the **minimum average waiting time** among all scheduling algorithms for a given set of processes. It schedules the process with the shortest CPU burst time first. FCFS can lead to the "convoy effect"; Round Robin is fair but not optimal for waiting time; Priority Scheduling may cause starvation.

---

**Q87.** Which of the following is an example of a deadlock prevention technique?

- A. Mutual exclusion
- B. Hold and wait
- C. Resource ordering
- D. Starvation

**Answer: C. Resource ordering**

> **Explanation:** Deadlock requires four conditions: Mutual Exclusion, Hold & Wait, No Preemption, and Circular Wait. **Resource ordering** (assigning a global order to resources and requiring processes to request resources in that order) prevents **Circular Wait**, thus preventing deadlock. Options A and B are actually *necessary conditions for deadlock*, not prevention techniques. Starvation is a different problem (indefinite waiting).

---

**Q88.** Which memory management technique divides memory into equal-sized blocks?

- A. Segmentation
- B. Paging
- C. Virtual Memory
- D. Swapping

**Answer: B. Paging**

> **Explanation:** **Paging** divides physical memory into fixed-size blocks called **frames** and logical memory into equal-sized blocks called **pages**. This eliminates external fragmentation. **Segmentation** divides memory into variable-sized segments based on logical units. **Virtual Memory** is a technique that allows execution of processes larger than physical memory. **Swapping** moves entire processes between RAM and disk.

### Section 9: Networking

---

**Q89.** Which layer of the OSI model is responsible for routing?

- A. Data Link Layer
- B. Network Layer
- C. Transport Layer
- D. Session Layer

**Answer: B. Network Layer**

> **Explanation:** The **Network Layer (Layer 3)** of the OSI model is responsible for **logical addressing and routing** — determining the best path for data packets to travel from source to destination across multiple networks. It uses **IP addresses** for addressing. The Data Link Layer handles frame delivery on the same network; the Transport Layer handles end-to-end communication.

**Q90.** Which protocol is used to securely transfer web pages?

- A. HTTP
- B. FTP
- C. HTTPS
- D. SMTP

**Answer: C. HTTPS**

> **Explanation:** **HTTPS (HyperText Transfer Protocol Secure)** is HTTP with **SSL/TLS encryption**, used to securely transfer web pages and protect data in transit. **HTTP** is the unsecured version; **FTP** is for file transfers; **SMTP** is for sending emails.

---

**Q91.** Which device connects multiple networks together?

- A. Hub
- B. Switch
- C. Router
- D. Repeater

**Answer: C. Router**

> **Explanation:** A **Router** operates at the **Network Layer (Layer 3)** and connects **multiple networks** together, routing packets between them using IP addresses. A **Hub** broadcasts data to all devices on the same network (Layer 1); a **Switch** connects devices within the same network (Layer 2); a **Repeater** regenerates signals to extend cable length.

---

**Q92.** Which addressing system works at the Data Link Layer?

- A. IP Address
- B. Port Number
- C. MAC Address
- D. URL

**Answer: C. MAC Address**

> **Explanation:** The **MAC (Media Access Control) Address** is a hardware address assigned to network interface cards (NICs) and operates at the **Data Link Layer (Layer 2)**. It is used for device-to-device communication on the same local network. **IP Address** works at Layer 3; **Port Number** works at Layer 4; **URL** is an application-layer concept.

---

**Q93.** Which protocol is connection-oriented?

- A. UDP
- B. TCP
- C. ICMP
- D. ARP

**Answer: B. TCP**

> **Explanation:** **TCP (Transmission Control Protocol)** is a **connection-oriented** protocol — it establishes a connection through a **three-way handshake** (SYN, SYN-ACK, ACK) before data transfer, ensuring reliable, ordered, and error-checked delivery. **UDP** is connectionless (no handshake, faster but unreliable); **ICMP** is used for error messages; **ARP** maps IP addresses to MAC addresses.

> - **Reduce phase**: Takes grouped key-value pairs and **aggregates/summarizes** them to produce the final output (e.g., counting word frequencies, summing values).
